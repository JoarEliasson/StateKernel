# Brand

StateKernel presents itself as a precise, engineering-first industrial tool.

## Direction

- Dark industrial-modern surfaces.
- Restrained, technical language.
- Measured visual hierarchy over marketing-heavy presentation.
- Strong emphasis on trust, instrumentation, and control.

## Core Palette

- Obsidian: `#0B0F14`
- Graphite: `#121923`
- Steel: `#1B2633`
- Signal Blue: `#4DA3FF`
- Process Teal: `#2FD1C4`

## Typography

- Primary UI typeface: IBM Plex Sans
- Monospace typeface: IBM Plex Mono

Brand assets live under `assets/brand/`.
