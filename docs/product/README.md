# Product

StateKernel is a local-first industrial developer tool for authoring deterministic OPC UA scenarios, comparing runtime and security profiles, and collecting benchmark-grade observability artifacts.

## Product Promise

The product is designed to help engineering teams validate OPC UA clients and workflows against repeatable, scenario-driven behavior instead of ad hoc scripts or simplistic demo servers.

## Primary Outcomes

- Model realistic industrial assets and scenarios.
- Run deterministic simulations through a dedicated runtime path.
- Compare secure and insecure endpoint profiles.
- Capture logs, metrics, and artifacts suitable for validation and benchmarking.

## Non-Goals

StateKernel is not currently positioned as a full digital twin platform, a historian, or a broad SaaS collaboration product.
